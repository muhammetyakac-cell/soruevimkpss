# Matematik Testi 350
Konu: Sayılar, Denklemler, Eşitsizlikler, Mutlak Değer
Süre: 20 Dakika

1. x, y, z pozitif tam sayılardır. 3x + 2y + 4z = 60 denkleminde x+y+z toplamının en büyük değeri kaçtır?
A) 25
B) 26
C) 27
D) 28
E) 29
Cevap: E

2. a ve b tam sayılar olmak üzere, |a-3| + |b+2| = 1 olduğuna göre a.b çarpımının alabileceği değerler toplamı kaçtır?
A) -12
B) -10
C) -8
D) -6
E) -4
Cevap: B

3. x < y < 0 < z olmak üzere, |x-y| + |z-x| - |y-z| ifadesinin eşiti nedir?
A) 2z-2y
B) 2z-2x
C) 2x-2y
D) 2z
E) 0
Cevap: B

4. 2x - 3 < 5x + 9 < 2x + 18 eşitsizliğini sağlayan kaç tane tam sayı değeri vardır?
A) 3
B) 4
C) 5
D) 6
E) 7
Cevap: C

5. x² - 4x + a = 0 denkleminin kökleri x1 ve x2'dir. x1² + x2² = 10 olduğuna göre a kaçtır?
A) 2
B) 3
C) 4
D) 5
E) 6
Cevap: B

6. a / (b+c) = b / (a+c) = c / (a+b) = k olduğuna göre k'nın alabileceği değerler çarpımı kaçtır?
A) -1
B) -0,5
C) 0
D) 0,5
E) 1
Cevap: A

7. |x² - 4| = |x - 2| denklemini sağlayan x değerlerinin toplamı kaçtır?
A) -1
B) 0
C) 1
D) 2
E) 3
Cevap: C

8. (x-2) / (x+3) >= 0 eşitsizliğini sağlamayan kaç tane negatif tam sayı vardır?
A) 1
B) 2
C) 3
D) 4
E) 5
Cevap: B

9. x, y gerçel sayılar olmak üzere -3 < x < 5 ve 2 < y < 6 olduğuna göre, 2x - 3y ifadesinin alabileceği en geniş değer aralığı nedir?
A) (-24, 4)
B) (-22, 4)
C) (-24, 0)
D) (-20, 2)
E) (-22, 6)
Cevap: A

10. x² + y² - 4x + 6y + 13 = 0 olduğuna göre x + y toplamı kaçtır?
A) -1
B) 0
C) 1
D) 2
E) 3
Cevap: A
