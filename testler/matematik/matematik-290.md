# KPSS Matematik Test 290
Süre: 20 Dakika
Konu: Sayılar, Denklemler, Eşitsizlikler, Mutlak Değer (Karma, Zor)

1. x ve y birer tam sayı olmak üzere, |x - 3| + |y + 2| = 1 olduğuna göre, x + y toplamının alabileceği değerler toplamı kaçtır?
A) -1 B) 0 C) 1 D) 2 E) 3
Cevap: B

2. a < b < 0 < c olmak üzere, |a - c| + |b - a| - |c - b| ifadesinin eşiti nedir?
A) -2a B) 2a C) 2c D) 2b E) 0
Cevap: A

3. 2 < |3x - 1| < 7 eşitsizliğini sağlayan x tam sayı değerlerinin toplamı kaçtır?
A) 0 B) 1 C) 2 D) 3 E) 4
Cevap: B

4. 3x - 2y = 7 ve 2x + y = 7 denklem sistemini sağlayan x ve y değerleri için x * y kaçtır?
A) 2 B) 3 C) 4 D) 5 E) 6
Cevap: B

5. a, b, c pozitif tam sayılar ve 2a = 3b = 4c olduğuna göre, a + b + c toplamının en küçük değeri kaçtır?
A) 12 B) 13 C) 18 D) 24 E) 26
Cevap: B

6. 5x + 3 > 2x - 9 eşitsizliğini sağlayan en küçük x tam sayısı kaçtır?
A) -3 B) -4 C) -5 D) -6 E) -7
Cevap: B

7. |x^2 - 4| = |x - 2| denklemini sağlayan x değerlerinin toplamı kaçtır?
A) -2 B) -1 C) 0 D) 1 E) 2
Cevap: C

8. (x - 2) / 3 <= (2x + 1) / 2 eşitsizliğini sağlayan x değerlerinin en geniş çözüm aralığı nedir?
A) x >= -7 B) x <= -7 C) x >= 7 D) x <= 7 E) x > -7
Cevap: A

9. Ardışık üç tek sayının toplamı 87 olduğuna göre, bu sayıların en büyüğü kaçtır?
A) 27 B) 29 C) 31 D) 33 E) 35
Cevap: C

10. x^2 - 5x + 6 = 0 denkleminin kökleri x1 ve x2 ise, x1^2 + x2^2 ifadesinin değeri kaçtır?
A) 10 B) 11 C) 12 D) 13 E) 14
Cevap: D
