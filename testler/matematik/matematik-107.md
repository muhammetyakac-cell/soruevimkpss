## Cevap Anahtarı

| Soru | Cevap |
|------|-------|
| 1 | D |
| 2 -> 175cm (2.5m = 250cm. 250 - 5*15 = 250 - 75 = 175) | D |
| 3 | B |
| 4 | A |
| 5 -> A: 10 saatte 100% (saatte 10%), 2 saatte 20% dolar. Kalan 80%. B: 15 saatte 100% (saatte 6.66%). 80 / 6.66 = 12 saat. Toplam 14 saat. | C |
| 6 -> Yol x. x/80 + x/120 = 5. (3x+2x)/240 = 5. 5x = 1200. x = 240. | C |
| 7 | D |
| 8 -> B = 3T. B+5 = 2(T+10). 3T+5 = 2T+20. T=15. B=45. | D |
| 9 -> Kesilen miktar x/5. Orta nokta x/2'den (x-x/5)/2'ye kayar. Kayma = x/2 - 2x/5 = x/10. x/10 = 10 -> x=100. | B |
| 10 -> n kişi, ortalama 20. Toplam = 20n. Yeni toplam 20n + 40. Yeni ortalama (20n+40)/(n+1) = 22. 20n+40 = 22n+22. 18 = 2n. n=9. | C |
